Welcome to Document Object Manipulation

This is a powerful JavaScript programming that allows programmers to access elements in the HTML document and having the power to add, delete or change the content of the HTML elements and attributes, and their content, and so on.

The HTML DOM is a standard object model and programming interface for HTML. It defines:

  1. The HTML elements as objects
  2. The properties of all HTML
  3. The methods to access all HTML elements
  4. The events of all HTML elements - Event listeners
